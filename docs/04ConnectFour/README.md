# Connect Four — Good vs Evil

Simple, polished Connect Four implementation with thematic visuals, drop physics, SVG arrow animations, and a victory modal.

Play locally: open `index.html` in your browser, or publish with GitHub Pages (see instructions below).

Key features
- Classic 7×6 Connect Four gameplay
- Gravity-drop animation, bounce effect, and SVG arrow highlights for 2/3/4-in-a-row
- Responsive layout, persistent score counters, and keyboard-friendly controls

Tech: HTML5, CSS3, Vanilla JavaScript (ES6+)

Run locally
1. cd into this folder
2. Open `index.html` in any modern browser

Quick GitHub publish steps
1. Create a repo on GitHub (or run `gh repo create YOURNAME/connect-four --public --source=. --push`)
2. From this folder run:

```bash
git init
git add .
git commit -m "Initial: Connect Four (Good vs Evil)"
git branch -M main
git remote add origin git@github.com:YOUR_GITHUB_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

3. In the repository Settings → Pages, set the source to `main` branch (root) and save.

Live URL (after Pages is enabled)
- https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/
